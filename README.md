# StrideLogic - Career Guidance AI

Grounded in Data, Ascending in Career.

StrideLogic is an intelligent career ascent platform designed to provide data-driven insights and personalized guidance for professional growth. This project leverages AI to help users navigate their career paths effectively.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
Open http://localhost:3000 with your browser to see the result.

Environment Variables
To run this project, you must create a .env.local file in your root directory with the following variables:

```text
DATABASE_URL=
GEMINI_API_KEY=
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
NEXT_PUBLIC_CLERK_SIGN_IN_URL=/sign-in
NEXT_PUBLIC_CLERK_SIGN_UP_URL=/sign-up
NEXT_PUBLIC_CLERK_AFTER_SIGN_IN_URL=/onboarding
NEXT_PUBLIC_CLERK_AFTER_SIGN_UP_URL=/onboarding
